# 全局安装

如果你只是想随便用下 VuePress，你可以在全局安装它：

# 全局安装
npm install -g vuepress

# 创建一个 markdown 文件
echo '# Hello VuePress' > README.md

# 开始编写文档
vuepress dev

# 构建
vuepress build
